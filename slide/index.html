Neural Network 101

Intro

A Neural Network is an Universal Approximator or, to say with different words, a flexible function that autonomously adapts its behaviour to satisfy as much as possible the relation between the number[s] passed to it as parameter[s] and the number[s] expected to be returned as relative result[s].

Neuron

  • Inputs (Dendrites)
  • Processor (Cell nucleus)
  • Output (Axon)

The neuron accumulates all incoming inputs, multiply by respective weights and sum all results. If it's in a certain threshold, it fires an output.

Activation function

  • Used to non-linear behavior.
  • Receive an output and normalizes it (between 0 and 1).
  • Usually it's a sigmoid function.

Feed-Foward Network

The output from input layer serves as an input for the next layer (hidden layer). It's repeated until reaching the final layer (output layer).

How a NN learn? TRAINING!

Backpropagation algorithm

Algorithm process:

  • Input -> Output
  • Then teach what should have been the correct output.
  • Adjusting the weights with the ideal output.
  • Start from output layer to input layer (reverse path).
  • Insert same input, but the weights are adjusted.
  • Repeat this process until the delta between ideal output and network output is small enough.

Gradient Descent

Calculate "slope" at current "x" position; Change x by the negative of the slope. (x = x - slope); Repeat until slope value is zero (slope == 0).

Training for XOR


						var synaptic = require("synaptic");

						// Create layers
						var inputLayer = new synaptic.Layer(2);
						var hiddenLayer1 = new synaptic.Layer(4);
						var hiddenLayer2 = new synaptic.Layer(4);
						var outputLayer = new synaptic.Layer(1);

						// Create weights connections
						inputLayer.project(hiddenLayer1);
						hiddenLayer1.project(hiddenLayer2);
						hiddenLayer2.project(outputLayer);
					

						// Assemble the network
						var myNetwork = new synaptic.Network({
						    input: inputLayer,
						    hidden: [
						        hiddenLayer1,
						        hiddenLayer2
						    ],
						    output: outputLayer
						});
					

						// Defining our trainer
						var trainer = new synaptic.Trainer(myNetwork);
						var trainingSet = [
							{ input: [0, 0], output: [0] },
							{ input: [0, 1], output: [1] },
							{ input: [1, 0], output: [1] },
							{ input: [1, 1], output: [0] }
						      
						];
					

						// Training!
						trainer.train(
						    trainingSet,
						    {
						        rate: .3, // Learning rate
						        iterations: 1000000, // Maximum number of Iterations
						        error: .0000005, // Minimum error
						        shuffle: true, // Shuffled after every iter.
						        cost: synaptic.Trainer.cost.MSE, // Self-explained
						        log: 1 // console.log the error and iter
						    }
						);
					

						// Test!
						console.log("[0, 0] => [" + myNetwork.activate([0, 0]) + "]");
						console.log("[0, 1] => [" + myNetwork.activate([0, 1]) + "]");
						console.log("[1, 0] => [" + myNetwork.activate([1, 0]) + "]");
						console.log("[1, 1] => [" + myNetwork.activate([1, 1]) + "]");

END
